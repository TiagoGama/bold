# Month Range Picker

O Month Range Picker permite que o usuário selecione um período em meses através de um calendário. Por padrão, o calendário apresenta sempre o ano atual. Caso já tenha uma data inserida, o calendário apresenta o ano referente à esta data.

## Boas práticas

- Você pode estabelecer um valor mínimo e máximo que o usuário poderá escolher.
- O campo deve permitir que o usuário também insira o valor através de input do teclado.
- Se o usuário deve selecionar um período com maior granularidade, como semanas ou dias, utilize o [Date Picker](/components/date-picker).
